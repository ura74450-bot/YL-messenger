# 💬 YL Messenger

Полноценный мессенджер: чаты, голосовые, фото/видео, файлы, статусы.

## Запуск локально

```bash
node server.js
# Открыть http://localhost:3001
```

## Публичный сервер (Render.com)

1. Загрузи папку на GitHub
2. render.com → New Web Service → выбери репозиторий
3. Start command: `node server.js`
4. После деплоя вставь URL в index.html:
   `const SERVER_URL = 'https://yl-messenger.onrender.com';`

## Функции

- Регистрация/вход с паролем, несколько аккаунтов
- Поиск по @никнейму
- Текст, фото, видео, файлы, голосовые
- Статусы (видео до 10 сек)
- Блокировка, очистка чата
- Онлайн-статус в реальном времени
- Аватарки сохраняются везде
